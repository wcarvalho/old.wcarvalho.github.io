Time Tracker System

January 08, 2016 by Wilka Carvalho

I present an overview of how to use the script in my post, “A Prescription for Managing and Tracking Your Time,” to track time delegation in Google Calendar using a Google spreadsheet

  1. Download this excel file, upload it into Google Sheets, and add this script to it.

    Open “Tools -> Script Editor…“
    Paste the contents of the script inside and save.

  2. Place the name of calendars you want to track vertically beginning at B8.
  3. If you want a calendar’s events to be used for tracking time assignment to a category, place text to the left of the calendar name (any text satisfies this).
  4. Place the name of categories you want to track horizantally beginning at B2.
  5. An event will contribute to a category’s time if the category is somewhere in the event’s title.
  6. Put the start date and end date across cells E7-H7 and E8-H8, respectively, in the format (date, month, year).
  7. Track your time using the Time Tracker menu which should appear as soon as you save the script.

Example:

Calendar: The calendar names are on the right side, while the category names are used in the event titles.
Spreadsheet: Time delegation for categories “570,” “561,” “research,” and “other” was only tracked using calendars “Work,” “Extra,” and “Personal.”
